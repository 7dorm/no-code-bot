# 🚀 Быстрый старт

## Установка и запуск

### 1. Установите зависимости (уже сделано)

```bash
npm install
```

### 2. Запустите приложение

```bash
npm run dev
```

Приложение откроется автоматически на `http://localhost:3000`

## 📖 Как использовать

### Создание проекта

1. **Создайте новый проект**: Нажмите кнопку "Создать" в тулбаре
2. Проект автоматически создастся и сохранится в localStorage

### Добавление блоков

1. **Откройте библиотеку блоков** (слева)
2. **Выберите тип блока** (Старт, Сообщение, Условие и т.д.)
3. Блок появится на canvas

### Редактирование блоков

1. **Кликните на блок** - чтобы выбрать его
2. **Нажмите Enter или E** - чтобы открыть редактор
3. **Или двойной клик** по блоку
4. Введите параметры и нажмите "Сохранить"

### Соединение блоков

1. **Наведите на выходной узел** блока (синяя точка снизу)
2. **Зажмите и перетащите** к входному узлу другого блока
3. Блоки соединены!

### Удаление блоков

1. **Выберите блок** кликом
2. **Нажмите Delete** или **Backspace**
3. Подтвердите удаление

### Настройки проекта

1. **Нажмите "⚙️ Настройки"** в тулбаре
2. **Укажите Telegram Bot Token** (получите у @BotFather)
3. **Добавьте глобальные константы** (если нужно)
4. **Нажмите "Применить и сохранить"**

### Сохранение проекта

- **"💾 Сохранить"** - скачивает JSON файл
- **Автосохранение** - в localStorage браузера
- **"📁 Загрузить"** - загружает проект из файла

### Предпросмотр

1. Создайте бота с хотя бы одним блоком **Старт**
2. Нажмите **"👁️ Предпросмотр"**
3. Протестируйте диалог в имитации чата

### Undo/Redo

- **"↶ Отменить"** или **Ctrl+Z** - отмена последнего действия
- **"↷ Вернуть"** - повтор действия

## ⌨️ Горячие клавиши

- **Delete / Backspace** - удалить выбранный блок
- **Enter / E** - открыть редактор выбранного блока
- **Escape** - закрыть модальные окна
- **Ctrl+Z** - отменить действие
- **Ctrl+Y** - вернуть действие

## 📱 Экспорт в Telegram

1. **Настройте проект** (укажите Telegram Token)
2. В будущем будет кнопка **"Экспорт для Telegram"**
3. Получите готовый код для деплоя

## 🎯 Типы блоков

- **▶️ Старт** - точка входа в диалог
- **💬 Сообщение** - отправка текста пользователю
- **🔀 Условие** - ветвление по условию (2 выхода: true/false)
- **📝 Переменная** - работа с переменными
- **🔌 API** - вызов внешнего API
- **📎 Файл** - обработка файлов
- **⏹️ Конец** - завершение диалога

## 🆘 Решение проблем

### Проект не сохраняется
- Проверьте, что localStorage не заблокирован
- Откройте DevTools → Application → Local Storage

### Блоки не соединяются
- Убедитесь, что перетаскиваете от выходного узла (синяя точка)
- Проверьте, что у блока есть доступный входной узел

### Предпросмотр не работает
- Убедитесь, что есть блок **Старт**
- Проверьте, что блоки правильно соединены

### Ошибка при загрузке проекта
- Проверьте формат JSON файла
- Убедитесь, что файл создан в этом редакторе
