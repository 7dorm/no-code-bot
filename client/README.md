# 🎨 Визуальный редактор чат-ботов

Веб-приложение для создания чат-ботов с визуальным интерфейсом drag-and-drop и интеграцией с Telegram.

## 📋 Возможности

### UC-1: Создание нового бота
- Создание пустого проекта с drag-and-drop интерфейсом
- Сохранение проектов локально

### UC-2: Настройки проекта
- Настройка Telegram токена
- Управление глобальными константами

### UC-3: Добавление блоков
- Создание блоков различных типов
- Ограничение на количество блоков (макс. 100)

### UC-4: Удаление блоков
- Удаление блоков с подтверждением
- Автоматическое удаление связей

### UC-5: Перемещение блоков
- Drag-and-drop функциональность
- Предотвращение выхода за границы экрана

### UC-6: Редактирование блоков
- Модальное окно для настройки параметров
- Валидация параметров

### UC-7: Условное ветвление
- Блоки условий с несколькими выходами
- Поддержка логических выражений

### UC-8: Соединение блоков
- Визуальные связи между блоками
- Автоматическое удаление старых связей (кроме conditions)

### UC-9: Удаление связей
- Удаление связей между блоками

### UC-10: Отмена действий (Undo)
- История до 50 действий
- Поддержка Ctrl+Z

### UC-11: Повтор действий (Redo)
- Возврат отмененных действий

### UC-12: Навигация по рабочему пространству
- Миникарта
- Управление масштабом и панорамированием

### UC-13: Сохранение проекта
- Экспорт в JSON формат
- Автосохранение в localStorage

### UC-14: Экспорт бота
- Сохранение конфигурации в файл
- Указание пути для сохранения

### UC-15: Импорт бота
- Загрузка проекта из файла
- Валидация формата

### UC-16: Интеграция с Telegram
- Генерация готового кода
- Создание архива с инструкцией

### UC-17: Предпросмотр бота
- Интерактивное тестирование
- Симуляция диалога

### UC-18: Использование реальным пользователем
- Работа бота в Telegram
- Обработка ошибок

## 🚀 Быстрый старт

### Установка

```bash
npm install
```

### Разработка

```bash
npm run dev
```

Приложение откроется на `http://localhost:3000`

### Сборка

```bash
npm run build
```

## 🛠 Технологии

- **React 18** - UI библиотека
- **TypeScript** - типизация
- **React Flow** - canvas для блоков
- **Zustand** - управление состоянием
- **Vite** - сборщик

## 📦 Структура проекта

```
src/
├── components/          # React компоненты
│   ├── Editor/         # Основной редактор
│   ├── Toolbar/        # Панель инструментов
│   ├── BlockLibrary/   # Библиотека блоков
│   ├── BlockEditor/    # Редактор блока
│   ├── Settings/       # Настройки проекта
│   └── Preview/        # Предпросмотр
├── store/              # Управление состоянием
├── types/              # TypeScript типы
└── utils/              # Утилиты (экспорт в Telegram)
```

## 🎯 Типы блоков

- **start** (▶️) - Стартовый блок
- **message** (💬) - Отправка сообщения
- **condition** (🔀) - Условное ветвление
- **variable** (📝) - Работа с переменными
- **api** (🔌) - API запросы
- **file** (📎) - Обработка файлов
- **end** (⏹️) - Конец диалога

## 💡 Использование

1. **Создание проекта**: Нажмите "Создать" в тулбаре
2. **Добавление блоков**: Выберите блок из библиотеки слева
3. **Соединение блоков**: Перетащите от выходного узла к входному
4. **Настройка блоков**: Кликните на блок для редактирования
5. **Сохранение**: Используйте кнопку "Сохранить" или автосохранение
6. **Предпросмотр**: Нажмите "Предпросмотр" для тестирования
7. **Экспорт**: Экспортируйте проект для Telegram

## 🔧 Настройки

В настройках проекта можно:
- Указать Telegram Bot Token от @BotFather
- Добавить глобальные константы
- Настроить параметры проекта
